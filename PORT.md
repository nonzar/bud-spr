# 接口文档

### 积分记录查询
返回指定openid\spr的积分结果。

**parameters**:
- **`类型`**
    - **`openid`**
    - **`spr`**
- **`关键字`**
- **`排序方式`**
- **`页码`**

**return**:
```javascript
//包含的部分字段
{
    openid:,
    产品名称:,
    产品积分:,
    积分时间:,
    渠道分类:,
    大区:,
    城市:,
    促销员id:,
    促销员:,
    PLT:
}
```
